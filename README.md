# Passive-Aggressive Email Detector

An on-prem MLOps application that scores emails for passive-aggression,
sarcasm, and overall tone; highlights problematic phrases; and offers a
constructive rewrite. Built as the DA5402 MLOps end-to-end project.

See `docs/ARCHITECTURE.md` and `../2026-04-26-passive-aggressive-email-detector-design.md`
for the full design.

## Quick start

```bash
cp .env.example .env
docker compose up -d
open https://localhost
```
